Generate Maze

Sadly there does not exist a UI to generate a maze. However, with the maze API server running,
make a request to POST http://127.0.0.1:5000/maze

JSON Body Required Fields
        # size - Object
        an object of {rows: n, cols: n} where n is the desired number of rows and columns

        # start - Array
        a two item array of [row, col]

        # goal - Array
        a two item array of [row, col]

        # algorithm - String
        a value of {"binary-tree" | "binary-tree-recur"}
Example Body
{
	"size": {
		"rows": 10,
		"cols": 10

	},
	"start": [0, 0],
	"goal": [9, 0],
	"algorithm": "binary-tree"
}
